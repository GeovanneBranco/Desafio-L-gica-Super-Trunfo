Este é um trabalho de faculdade onde o proposita é entender e aplicar a logica de progamacao em C, o meu objetivo é entregar um jogo de SuperTrunfo onde o usuario escolha atributos da sua carta e o progama faca um comparativo do atributo selecionado com um adversario, no caso o proprio progama simula um adversario atraves de um gerador de numeros aleatorios,tomei a liberdade de criar um pequena variedade de cartas pre-determinadas, onde o jogador seleciona uma, e o programa vai selecionar uma diferente.

 Atraves de comandos de switchs o usuario vai poder selecionar sua carta e atributos, o programa vai ler os dados das variaveis atrelados a carta e calcular outras variaveis como a densidade demografica,PIB per capita, e o atributo especial o SuperPoder.

 O resultado final é mostrar ao usuario os dados de ambas as cartas e quem foi o vencedor com base logica dos atributos, onde o maior vence com excessao da densidade demografica onde o menor vence.
